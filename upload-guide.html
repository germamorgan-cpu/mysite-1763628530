Guide d'Upload des Fichiers Excel

Comment intégrer vos fichiers Excel dans QualXP

Méthodes disponibles

1. Upload direct via FTP/File Manager

Recommandé pour les fichiers existants

  • Créez un dossier "excel/" à la racine de votre site
  • Organisez les fichiers par catégories :
    • excel/controle-reception/
    • excel/plan-experience/
    • excel/validation-methodes/
    • excel/metrologie/
    • excel/cartes-controle/
    • excel/test-divers/
  • Nommez les fichiers de façon explicite

2. Stockage Cloud (Google Drive, OneDrive, Dropbox)

Solution flexible et sécurisée

Partagez chaque fichier avec le lien "Toute personne disposant du lien peut visualiser"

Remplacez les liens "#" dans index.html par vos liens de partage
